What is CardioML?

CardioML is a machine learning-powered web application designed to predict the risk of heart disease based on patient biomarkers and vital signs. The model uses Logistic Regression trained on clinical data to provide risk assessments.

Model Performance

  • Algorithm: Logistic Regression
  • Test Accuracy: 86.05%
  • Training Data: 1,177 patients
  • Features Used: 9 high-correlation biomarkers

Features Used

The model uses these 9 key indicators:

  • Heart Rate (bpm)
  • RDW - Red Cell Distribution Width (%)
  • Leucocyte Count (K/ÂµL)
  • PT - Prothrombin Time (s)
  • INR - International Normalized Ratio
  • Urea Nitrogen (mg/dL)
  • Blood Potassium (mEq/L)
  • Anion Gap (mEq/L)
  • Lactic Acid (mmol/L)
