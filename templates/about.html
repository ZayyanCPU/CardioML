About CardioML

A machine learning project for heart disease prediction

Project Overview

CardioML is a machine learning project focused on predicting heart disease outcomes using clinical and laboratory parameters. This project demonstrates a complete data science workflow from exploratory data analysis through model deployment, comparing multiple classification algorithms to identify the optimal predictive model.

Methodology

The project follows a rigorous machine learning pipeline:

  • Data Preprocessing: Handling missing values with SimpleImputer
  • Feature Selection: Using correlation analysis (threshold > 0.129)
  • Feature Scaling: StandardScaler normalization
  • Model Training: 60-20-20 train-validation-test split
  • Hyperparameter Tuning: GridSearchCV with 5-fold cross-validation
  • Evaluation: Accuracy, AUC-ROC metrics
